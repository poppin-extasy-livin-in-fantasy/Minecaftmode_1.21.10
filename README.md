# Minecraftmod 1.21.10

My first Minecraft Forge mod for version 1.21.10, written in Java.

## Features

### Blocks
- **Tetra Block** — polished block, strength 4, requires correct tool, amethyst sound
- **Raw Tetra Block** — raw variant, strength 3, requires correct tool

### Items
- **Tetra Gem** — crafted material
- **Raw Tetra Gem** — raw variant obtained from ore
- **Chisel** — custom tool with 32 durability that transforms blocks on right-click:
  - Stone → Stone Bricks
  - End Stone → End Stone Bricks
  - Deepslate → Deepslate Bricks
  - Iron Block → Diamond Block
  - Dirt → Tetra Block

### Other
- Custom creative tab
- Russian localization
- Smelting recipe for Raw Tetra Gem → Tetra Gem

## Built with

- Java
- Minecraft Forge 1.21.10
- Gradle
