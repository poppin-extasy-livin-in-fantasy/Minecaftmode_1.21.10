# Minecraftmod 1.21.10

A Minecraft Forge mod for version 1.21.10, my first mod project.

## Features

- Custom ore block with smelting recipe
- Custom item with unique functionality  
- Custom creative tab
- Crafting recipes

## Built with

- Java
- Minecraft Forge 1.21.10
- Gradle

## Status

Early learning project — first mod, work in progress.
